Wishbone

Wishbone is a Backbone application that was made in three weeks for The Iron Yard's final-project demonstration. My goal was to develop an application that helped users keep track of the products that they want but also keep track of the products that their friends are interested in.

    The assignment required that apps:
  • used an MVC framework
  • communicated with at least one HTTP API
  • featured CRUD functionality
  • included User Authorization and Authentication
  • adapted repsonsively to mobile and desktop
  • showcased the ability to plan and execute the production of a Front-End application

The Three Broomsticks
